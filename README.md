# modus6 Astro Website

A clean Astro launch site for modus6, designed to deploy on Netlify and integrate with Go High Level for forms, calendar, CRM, and workflows.

## Local setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Netlify

Build command: `npm run build`  
Publish directory: `dist`

## Pages

- `/`
- `/framework`
- `/readiness-audit`
- `/implementation`
- `/resources`
- `/blog`
- `/contact`

## GHL embeds

Embeds are in `src/components/GHLEmbeds.astro`.
